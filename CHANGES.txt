🎯 完整的自动交易流程
================================================================================

启动 → 信号分析 → 交易执行 → 日志记录 → 通知推送

具体步骤:
  1. main.py 启动
  2. ExecutionFlow 初始化
  3. strategy.analyze() 生成信号
  4. 根据信号调用 executor 的对应方法:
     - open_long() → 开多仓
     - open_short() → 开空仓
     - close_position() → 平仓
     - adjust_stop_loss() → 调整止损
  5. 记录到 execution_log.json
  6. 推送 Telegram 通知
